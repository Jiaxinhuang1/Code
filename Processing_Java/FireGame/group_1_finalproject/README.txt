1. Download all the processing files including the classes and the main program

2. Put all the files within one folder named group_1_finalproject

3. Launch group_1_finalproject.pde and make sure the other classes are opened too.

4. Click Sketch -> Import Library -> Add Library -> Look for Sound -> Install if you havent already.

4. Click Run

5. Interact with the GUI buttons using mouse click in the screens

6. Change volume by playing with the volume slider or mute button. Sound can be unmuted if press unmute then back

6. Move fire ball using  Arrows or WASD (collect fireballs and avoid water traps and enemies)

7. Pause in gameplay using P button

8. Reach the goal to see leaderboard.
