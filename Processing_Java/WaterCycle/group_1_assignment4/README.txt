1. Download all the processing files (group_1_assignment4, Arrows, Cloud, Evaporation, Fish, Precipitation, Rain, RunOff, Sun) and the sound file (Water Cycle).

2. Put all the files within one folder named group_1_assignment4

3. Launch group_1_assignment4.pde and make sure the other classes are opened too.

4. Go to sketch --> import library --> add library then scroll down until you find the Sound package and click install

5. When the sound package is installed, click Run
